# Rolldown Workspace

This monorepo hosts the `@calmdown/rolldown-workspace` utility as well as
several Rollup/Rolldown plugins:

|||
|---|---|
| [rolldown-workspace](./packages/rolldown-workspace/) | pipelining utility for Rolldown in workspaces |
| [rollup-plugin-copy](./packages/rollup-plugin-copy/) | copy files during builds |
| [rollup-plugin-delete](./packages/rollup-plugin-delete/) | delete files during builds |
| [rollup-plugin-lightningcss](./packages/rollup-plugin-lightningcss/) | load, transform and bundle CSS |
| [rollup-plugin-shell](./packages/rollup-plugin-shell/) | run shell commands during builds |
| [rollup-plugin-text-loader](./packages/rollup-plugin-text-loader/) | load file contents as runtime constants |
| [rollup-plugin-watch](./packages/rollup-plugin-watch/) | watch additional files in watch mode |
